# FinanceFlow 💰

A modern personal finance dashboard built with React, TypeScript, Tailwind CSS, and Zustand.

## Features

- Dashboard overview
- Add transactions
- Recent transactions table
- Responsive layout
- State management with Zustand

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Zustand
- Vite

## Run Locally

```bash
npm install
npm run dev
```
